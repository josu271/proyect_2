# 1\. Resumen del Proyecto

El proyecto 'Sistema Web de Generación Óptima de Horarios Académicos' fue desarrollado durante el ciclo académico del curso Taller de Proyectos 2, con el objetivo de crear una solución web inteligente capaz de generar automáticamente horarios académicos universitarios utilizando técnicas de Problemas de Satisfacción de Restricciones (CSP) mediante Google OR-Tools.

El sistema aborda el problema NP-Hard de la planificación de horarios, reduciendo conflictos, optimizando el uso de aulas y docentes, y automatizando un proceso que tradicionalmente se realizaba de forma manual. Se desarrolló con una arquitectura moderna fullstack: FastAPI + PostgreSQL en el backend y React + Vite en el frontend, con autenticación JWT y 4 roles diferenciados.

El proyecto fue completado exitosamente, cumpliendo con los criterios de aceptación establecidos: funcionalidades implementadas, motor CSP operativo, cobertura de pruebas superior al 90%, y documentación completa.
